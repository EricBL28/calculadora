# Calculadora

Nombre del autor: Eric Antonio Bermeo Leon

## Explicacion de Incio de Apliacion Calculadora

1. Intalar los paquetes necesarios para iniciar el proyecto 

    ```bash
    npm install
    ```
2. Para iniciar el proyecto 

    ```bash
    npm run start
    ```
